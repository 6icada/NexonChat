NexonChat is simple tcp chat.

How to use NexonChat:

If you want to open server on linux use "Server_Linux".
If you want to open server on windows use "Server_Windows".

If you want to use client on linux use "Client_Linux".
If you want to use client on windows use "Client_Windows".

If you want to use controler server on linux use "ControlerServer_Linux".
If you want to use controler server on windows use "ControlerServer_Windows".

If you want to port forward server use "Port_Forwarder".

"Server_Linux" && "Server_Windows" vs "ControlerServer_Linux" && "ControlerServer_Windows"

When you are using "Server_Linux" or "Server_Windows" only one client can join server.But when you are using "ControlerServer_Linux" or "ControlerServer_Windows" multiple clients can join server.

Port forward guide:

When you port forward with ngrok client must type forward IP and forward port.For example:
NonForwarded:
  IP: 192.168.100.4
  Port: 4444

Forwarded:
  IP: 0.tcp.ngrok.io
  Port: 12846
